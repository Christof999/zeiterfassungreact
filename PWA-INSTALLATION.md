# PWA Installation - Lauffer Zeiterfassung

## App-Installation auf verschiedenen Geräten

### Android (Chrome/Edge/Firefox)
1. Öffnen Sie die Website in Chrome, Edge oder Firefox
2. Tippen Sie auf das Menü (drei Punkte)
3. Wählen Sie "App installieren" oder "Zum Startbildschirm hinzufügen"
4. Folgen Sie den Anweisungen

### iPhone/iPad (Safari)
1. Öffnen Sie die Website in Safari
2. Tippen Sie auf das Teilen-Symbol (Pfeil nach oben)
3. Scrollen Sie nach unten und wählen Sie "Zum Home-Bildschirm"
4. Tippen Sie auf "Hinzufügen"

### Desktop (Chrome/Edge)
1. Öffnen Sie die Website in Chrome oder Edge
2. Klicken Sie auf das Install-Symbol in der Adressleiste
3. Oder nutzen Sie den "App installieren" Button auf der Website
4. Folgen Sie den Anweisungen

## Features der installierten App
- ✅ Funktioniert offline (eingeschränkt)
- ✅ Schneller Start vom Homescreen
- ✅ Vollbild-Erfahrung ohne Browser-UI
- ✅ Push-Benachrichtigungen (zukünftig)
- ✅ Automatische Updates
- ✅ Native App-Gefühl

## Systemanforderungen
- HTTPS-Verbindung erforderlich
- Moderne Browser (Chrome 67+, Safari 11.1+, Firefox 67+, Edge 79+)
- Unterstützte Geräte: Android, iOS, Windows, macOS, Linux
